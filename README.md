# Task Management Application

A Django-based Task Management system with role-based access control (SuperAdmin, Admin, User) and JWT authentication.

## Features

- **Role-based access**: SuperAdmin, Admin, and User roles
- **JWT Authentication**: API access with JSON Web Tokens
- **Task Management**: Create, update, delete tasks
- **Completion Reports**: Submit reports with worked hours
- **Admin Panel**: Separate dashboards for SuperAdmin and Admin
